parseIntRange

fun String.parseIntRange(): IntRange?

Parses this string as IntRange. The format is equal to Kotlin rangeTo operator. Example usage:

val range = "56..343".parseIntRange()

println(range?.start) // 56
println(range?.endInclusive) // 343
